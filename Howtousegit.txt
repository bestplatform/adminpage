git을 사용하기 위해서는 user.email과 user.name을 등록해야 합니다
$ git config --global user.email "<이메일 주소>"
$ git config --global user.name "<이름>"
echo는 리눅스 명령어 입니다. README.md 파일에 # my_app 를 삽입합니다. README.md파일이 없는 경우 생성하고 삽입합니다.
echo # my_app >> README.md
git init
git remote add origin http://github.com/imtaekh/my_app.git
온라인 저장소를 설정합니다.(origin을 해당 주소로 설정)

git add .
git commit -m "<commit message>"
git push -u origin master

git pull

git clone + <해당 github주소>를 하면 해당 온라인 저장소가 내 컴퓨터로 다운로드 됩니다

git log
모든 commit들과 디테일을 보여줍니다.
git log --oneline
commit id의 일부와 commit message만 보여줍니다.
git reset <commit id> --hard
해당 commit으로 돌아갑니다.
